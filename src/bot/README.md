### `Commands`
Esta pasta abriga a implementação de todas as funcionalidades realizadas pelo bot inerentes ao grupo COMPET exclusivamente tais como adicionar um novo membro a equipe, registrar frequência dentre outras e, seguindo a estrutura para presente no arquivo `Command.ts` na pasta `/src/bot/structures`. Dessa forma temos para cada comando o seu nome, a sua descrição que é sua funcionalidade e, em `run` o que o mesmo responde após ser executado. Seguem os seguintes comandos:

- `ata de reunião`: responde com o link do formulário da ata de reunião
- `novo membro(create)`: adiciona um novo membro a equipe COMPET
- `justificar frequência`: responde com o link do formulário para justificar ausência
- `remover membro(quitMember)`: remove um membro da equipe COMPET
- `registrar frequência`: responde com o link do formulário para registro de frequência
- `remover membro(quitMember)`: remove um membro da equipe COMPET
- `tornar scrum(turnIntoScrum)`: responsável por tornar um membro Scrum Master do COMPET
- `remover membro(quitMember)`: remove um membro da equipe COMPET
- `remover membro(quitMember)`: remove um membro da equipe COMPET
